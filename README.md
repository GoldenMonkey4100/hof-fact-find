# HOF Broker Fact Find

Professional fact find form with Mercury Nexus API integration.

## Quick Deploy to Vercel

1. Push this repository to GitHub
2. Import to Vercel
3. Add environment variables:
   - `MERCURY_API_KEY` - Your Mercury API Key
   - `MERCURY_API_TOKEN` - Your Mercury API Token  
   - `MERCURY_ENV` - Set to `sandbox` for testing
   - `VITE_MERCURY_API_URL` - Set to `https://uatapis.connective.com.au/mercury/v1`
4. Deploy!

## Local Development

```bash
npm install
npm run dev
```
